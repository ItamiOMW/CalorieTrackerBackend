# CalorieTrackerBackend

Ktor Backend app for CalorieTracker android app.
[Android app](https://github.com/ItamiOWM/CalorieTracker)

## Tech Stack:

 - [Ktor](https://ktor.io/)

 - [Koin](https://insert-koin.io/docs/reference/koin-ktor/ktor/)

 - [JBCrypt](https://mvnrepository.com/artifact/org.mindrot/jbcrypt)

 - [Google Api Client](https://developers.google.com/api-client-library/java)

 - [Postgresql](https://www.postgresql.org/)

 - [Exposed](https://ktor.io/docs/interactive-website-add-persistence.html)

 - [HikariCP](https://mvnrepository.com/artifact/com.zaxxer/HikariCP)

 - [Docker](https://www.docker.com/)
